# -Final_Project_Excel
# Sales Data Analysis & Dashboard Project

## Project Overview

This project is an Excel-based Sales Data Analysis and Dashboard system created to perform data cleaning, business analysis, forecasting, reporting, and visualization using advanced Excel features.

The workbook includes:

* Data analysis tasks
* What-If Analysis
* Linear Regression
* Pivot Tables
* Dashboard Visualization
* Customer Insights
* Formula-based analysis

File Name: `Final_Project.xlsx`

---

| Sheet Name            | Description                                           |

| Final Project Dataset | Main raw dataset used for analysis                    |

# Features Implemented

## 1. Data Cleaning

* Removed duplicates
* Fixed formatting issues
* Organized dataset structure
* Applied Excel formatting

## 2. Formula & Function Analysis

Used multiple Excel formulas including:

* `SUMIF`
* `COUNTIF`
* `VLOOKUP`
* `IF`
* `LEFT`
* `RIGHT`
* `TEXT`
* `CONCAT`
* `AVERAGE`
* `MAX`
* `MIN`

## 3. What-If Analysis

Implemented:

* Goal Seek
* Scenario Manager

Used to analyze the impact of changing discounts and sales values on profit.

## 4. Linear Regression

Performed regression analysis using Excel Analysis ToolPak to identify relationships between:

* Sales
* Profit

## 5. Descriptive Statistics

Generated statistical summaries including:

* Mean
* Median
* Standard Deviation
* Minimum
* Maximum
* Range

## 6. High-Value Customer Analysis

Used GroupBy-style calculations and formulas to identify:

* Customers with highest purchase value
* Frequent buyers
* Revenue contribution

## 7. Frequently Purchased Product Analysis

Identified:

* Most purchased products
* Repeating customer-product combinations

## 8. Dashboard Creation

Created an interactive dashboard with:

* KPI Cards
* Charts
* Sales insights
* Profit tracking
* Category analysis
* Regional analysis
